# Interactive Diatom Experience

An engaging three-stage interactive experience featuring diatoms.

## Setup

1. Add your diatom images to the `images/` directory:
   - `images/diatom1.png`
   - `images/diatom2.png`
   - `images/diatom3.png`
   - `images/diatom4.png`

2. Open `index.html` in a web browser

## How It Works

### Stage 1: Selection
- Four diatom images are displayed on the screen
- Click on any diatom to select it
- The other three diatoms will fade away and disappear

### Stage 2: Guide the Diatom
- A white circle appears in the lower left corner of the screen
- Your selected diatom now follows your mouse cursor
- Guide the diatom to touch the white circle to complete the stage

### Stage 3: Journey Complete
- A completion message is displayed
- Your selected diatom is shown in its final form

## Features

- Smooth animations and transitions
- Responsive mouse tracking
- Collision detection between diatom and target circle
- Beautiful gradient background
- Glass-morphism UI elements

## Technologies Used

- HTML5
- CSS3 (with animations and modern effects)
- Vanilla JavaScript

Enjoy your journey with the diatoms!
